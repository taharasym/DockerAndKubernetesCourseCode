Creating a Deployment

Also the same as the pod yaml file with extra stuff

apiversion: has a leading apps instead of v1 so app/v1

metadata about the deployment
spec
  selector
    matchLabels  <== match the label

livenessProbe
readnessProbe

kubectl commands with deployments

k create -f nginx.deployment.yml --save-config

k describe [pod | deployment] [pod-name | deployment-name]

k apply -f nginx.deployment.yml

k get deployments --show-labels

k get deployments -l app=my-nginx

k scale -f nginx.deployment.yml --replicas=

k scale deployment [deployment-name] --replicas=4
