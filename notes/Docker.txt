Containerizing Apps

Each part of the microservices can be fixed, worked on independently in a container, they all talk to each other to form that useful app.


# build the image out of all the files in this directory and below, 

docker image build -t nigelpoulton/ctr-demo:2 .

# normally we push this image to a registry or you can push it to a Docker Hub

docker image push nigelpoulton/ctr-demo

# Run this image we just built
docker container run -d --name web -p 8000:8080 nigelpoulton/ctr-demo:2
