Rolling Update Deployment

Deploy all the yaml file in the current directory

k create -f ./ --save-config   <=== let us know what we started with

1. Update the image to a newer image version and save

  nginx:1.17.8-alpine 

2. Run the curl command to see that there is no down time in another terminal, it will run 100 times

  bash curl-loop.sh or sh curl-loop.sh

2. Run the deployment

 k apply -f nginx.deployment.yml 

 deployment.apps/my-nginx configured


3. View the status of the rollout

 k rollout status deployment my-nginx

4. Shell into one of the containers

k get pod
k describe pod my-nginx-66dd4b8769-6757l

5. clean up

k delete -f ./   <== this will run both the deploy and services yaml files
